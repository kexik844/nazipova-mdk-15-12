# Интернет-магазин на Django

Простой интернет-магазин с каталогом товаров, корзиной и системой заказов.

## Особенности

- Каталог товаров с категориями
- Поиск и фильтрация товаров
- Корзина на сессиях (работает для гостей)
- Система заказов с уменьшением остатков
- Личный кабинет для пользователей
- Админ-панель для управления
- Адаптивный дизайн с Tailwind CSS
- Фиолетово-розовая цветовая схема

## Технологии

- Python 3.8+
- Django 4.2+
- SQLite (для разработки)
- Tailwind CSS
- HTML5
